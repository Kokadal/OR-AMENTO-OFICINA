# Integração API Placas

Incluído:
- Consulta real da placa via https://wdapi2.com.br/consulta/PLACA/TOKEN
- Token API Placas em Configurações > Alterar dados oficina
- Preenchimento automático de placa, marca, modelo, ano, cor, FIPE quando retornado
- Fallback manual em caso de erro

Atenção: como o app está em GitHub Pages/frontend, o token pode ser visto no código. Para produção, usar backend/proxy seguro.
